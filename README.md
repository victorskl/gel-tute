# gel-tute

(formally EdgeDB)

## CLI

Use Homebrew on macOS.

https://docs.geldata.com/reference/using/cli

### Remove legacy
```
brew uninstall edgedb-cli
brew untap edgedb/tap
```

### Install gel
```
brew tap geldata/tap
brew install gel-cli
```

```
which gel
gel --version
gel --help
```
